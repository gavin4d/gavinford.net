Custom Printed Circuit Board

Schematic

The schematic has all of the components and connections represented as symbols and lines. This is how you will make your circuits without worrying how they will be laid out on the PCB.

In the schematic, you should be able to import a symbol for each of your components. KiCAD has a built-in library of many common chips, but you can import others if needed. The component symbol will have all the output pins that you can wire together with other components.

In each component’s data sheet there will likely be an application circuit. For most applications, this circuit will work perfectly and can be copied directly from the datasheet. Most components will require decoupling capacitors on the voltage input pin. These capacitors reduce the amount of noise on the power rails.

Layout

The PCB layout is where the actual copper of the board can be created. You can arrange all of your components on the board and start wiring them together. Make sure to add a flood fill of copper on all board layers and connect this to ground. This will fill all extra space on the board with copper, which provides the signals with a ground plane voltage reference.

When doing the layout make sure to put capacitors as close to the chip’s power pins as possible, and keep high frequency signals as direct as possible.
